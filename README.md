# 🔥 RoastMyRoom

Your room. Brutally judged.

RoastMyRoom is an AI-powered web app that analyzes room photos and delivers hilarious, personality-driven roasts through customizable mascots.

Upload a room photo, choose your favorite mascot, and get a Drip Score, style analysis, roast report, and improvement suggestions.

## Features

* 📸 Upload room photos
* 🤖 Multiple AI mascots

  * Room Goblin
  * Dumpster Dan
  * Judge Cat
  * Chaos Crow
* 🔥 Funny AI-generated roasts
* 📊 Drip Score system
* 🎨 Room style detection
* 💡 Practical room improvement tips
* 📱 Fully responsive design
* 🌙 Modern dark-themed UI

## Tech Stack

* React
* Vite
* Tailwind CSS
* AI Vision Analysis
* Vercel Deployment
